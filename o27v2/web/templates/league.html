Methodology
Pythagorean win % uses the Bill James exponent of 1.83, fitted to MLB. O27 has a higher run environment so the absolute exponent isn't ideal, but the residual (Pythag W minus actual W) still flags teams winning more or fewer games than their run differential implies.

Team OPS+ / OPS / wERA / xFIP are computed by aggregating every per-player game row into a single team-level row and running it through the same sabermetric pipeline a player gets. This is "team-quality on the WAR scale," not "sum of individual player WARs" — the team-WAR figure won't match a roster-WAR sum.

Distributions use linear-interpolated quartiles (matches numpy's default percentile()). The σ column is population standard deviation across all 30 teams.

Outlier z-scores use the same population σ. A z above 1.5 is notable; above 2.0 the team is genuinely tail-end on that metric.
