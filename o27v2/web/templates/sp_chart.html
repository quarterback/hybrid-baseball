How to read it. Starters are O27's projected arms — the pitcher the sim would hand the ball to given each club's rest as of this date. Like an MLB probable, the pick can shift once intervening games run.

Rank is a value score combining the pitcher's season skill (K-BB%, run prevention, WHIP) with the matchup (the opponent's team offense). The 10 / 12 / 15+ columns are start recommendations for shallow / medium / deep leagues — an x means "start him." Deep includes everyone shallow and medium does, never fewer.

Opp Off ranks the opposing team's offense by wOBA across the league (1 = toughest offense to face). It is a team-overall measure; a split by the pitcher's handedness is a future enhancement.
