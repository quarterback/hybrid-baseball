Build a Peer-League Universe

Several co-equal major leagues in one world (the basketball-universe model) — each its own size, home region and playing style. Leagues run fully independently (no interleague games); players cross over via transfers and the offseason. Each league's style sets the talent the generator produces, so the leagues play measurably different baseball without any rule changes.
