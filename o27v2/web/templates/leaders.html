Methodology
Batting rates are PA-denominated. PAVG = H/PA; OPS+ / wOBA+ are league-relativized (100 = league avg, higher = better). Stay% is the rate at which a batter chose stay over run on a contact event with runners on — the load-bearing O27 mechanic.

Pitching uses three result-tier metrics that each tell a different story. wERA weights earned runs by arc position (outs 1–9 ×0.85, 10–18 ×1.00, 19–27 ×1.20) and league-anchors to raw ER/27. xFIP strips out batted-ball luck and defense — (13·HR + 3·BB − 2·(K + foul-outs)) per 27 outs, re-fit each render so league xFIP = league wERA. Decay measures late-arc fade (K%_arc1 − K%_arc3 in points; lower = more durable). GSc is a per-appearance Game Score (50 = neutral, 100 = perfection). OS+ is league-relative outs per appearance (100 = avg). PA-rate K% counts foul-outs as Ks. WAR uses an O27-fitted runs-per-win factor (~21 vs MLB's ~10). W/L credits an SP with ≥ 12 outs and the lead, otherwise the most-effective reliever.
