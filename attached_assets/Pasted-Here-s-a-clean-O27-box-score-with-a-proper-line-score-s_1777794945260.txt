This is what a clean O27 super-inning box score should look like. Three things to note for the agent:

**Phase separation.** Each phase (regulation, SI1, SI2, SI3) gets its own batting and pitching table. Game totals at the bottom roll everything up per-player.

**OS% bounded at 100%.** In each phase, OS% is calculated against that phase's out cap (27 for regulation, 5 for super-innings). Never exceeds 100%. Game-level OS% can be computed against total game outs if you want it on the player page.

**Line score at top.** Reg / SI1 / SI2 / SI3 / Total. Anyone reading the box score can see exactly when scoring happened. Standard cricket Super-Over format adapted for baseball.

**Unattributed outs reconciled in Game Notes**, not patched into the totals row with a "legacy" label. The note says "5 unattributed outs on baserunning/FC" and the math reconciles cleanly.
